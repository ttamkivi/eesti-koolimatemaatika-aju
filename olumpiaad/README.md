# Matemaatikaolümpiaad

Eesti matemaatikaolümpiaadi juhend, ettevalmistusmaterjal ja näidisülesanded.

## Kavandatud sisu

- `juhend.md` — kuidas valmistuda, mis tasemed on, millal toimub
- `ettevalmistus.md` — teemad ja oskused, mida oodatakse
- `naidisulesanded.md` — kommenteeritud varasemate aastate ülesanded
- `lingid.md` — Eesti MO ametlik leht, arhiivid

Allikad: [matemaatika.edu.ee](https://www.matemaatika.edu.ee/), Eesti Matemaatika Selts.
