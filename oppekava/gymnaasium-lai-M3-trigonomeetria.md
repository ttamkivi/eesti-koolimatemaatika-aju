## Olulised seosed

**Ühikringi definitsioon:**
$$\sin^2 \alpha + \cos^2 \alpha = 1$$

**Trigonomeetriline ühik (fundamentaalne identiteet):**
$$\sin \alpha = \frac{y}{r}, \quad \cos \alpha = \frac{x}{r}, \quad \tan \alpha = \frac{\sin \alpha}{\cos \alpha}$$

**Liitmisvalemid:**
$$\sin(\alpha \pm \beta) = \sin \alpha \cos \beta \pm \cos \alpha \sin \beta$$
$$\cos(\alpha \pm \beta) = \cos \alpha \cos \beta \mp \sin \alpha \sin \beta$$

**Kahekordse nurga valemid:**
$$\sin 2\alpha = 2 \sin \alpha \cos \alpha$$
$$\cos 2\alpha = \cos^2 \alpha - \sin^2 \alpha$$

**Trigonomeetriliste võrrandite lahendid:**
$$\sin x = a \Rightarrow x = (-1)^n \arcsin a + \pi n, \quad n \in \mathbb{Z}$$
$$\cos x = a \Rightarrow x = \pm \arccos a + 2\pi n, \quad n \in \mathbb{Z}$$

## Õppesisu

### Ühikring ja trigonomeetrilised funktsioonid
Ühikringi mõiste, siinuse ja koosinuse geomeetriline defineerimine ühikringil. Tangens ja kotangens. Nurga radiaanmõõt ja astmemõõt. Erinevate nurkade trigonomeetriliste funktsioonide väärtused.

### Trigonomeetrilised identiteedid
Fundamentaalne trigonomeetriline identiteet. Redutseerimisvalemid. Liitmisvalemid ja neist tuletuvad kahekordse nurga valemid. Summa teisendamine korrutiseks ja vastupidi.

### Trigonomeetriliste võrrandite lahendamine
Lihtsamate võrrandite lahendamine ($\sin x = a$, $\cos x = a$, $\tan x = a$). Trigonomeetriliste võrrandite lahendamine identiteetide ja faktoriseerimise abil. Abinurga kasutamine.

### Trigonomeetria rakendused
Kolmnurga lahendamine (siinuste ja koosinuste teoreem). Kauguste ja nurkade arvutamine praktikas. Perioodilised nähtused ja nende modelleerimine trigonomeetriliste funktsioonidega.

### Trigonomeetriliste funktsioonide graafikud
Siinuse, koosinuse ja tangensi funktsioonide graafikud. Funktsioonide omadused: perioodilisus, paarsus/paaritus, ekstremaalsed väärtused. Graafiliste teisenduste rakendamine ($y = a \sin(bx + c) + d$).
