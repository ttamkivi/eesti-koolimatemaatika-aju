## Olulised seosed

**Astmete omadused:**
$$a^m \cdot a^n = a^{m+n}, \quad (a^m)^n = a^{mn}, \quad (ab)^n = a^n b^n$$

**Ratsionaalne aste:**
$$a^{\frac{m}{n}} = \sqrt[n]{a^m}$$

**Eksponentfunktsiooni omadused:**
$$a^x \cdot a^y = a^{x+y}, \quad a^{x-y} = \frac{a^x}{a^y}$$

**Logaritmi definitsioon:**
$$\log_a b = c \Leftrightarrow a^c = b$$

**Logaritmide omadused:**
$$\log_a (xy) = \log_a x + \log_a y$$
$$\log_a \frac{x}{y} = \log_a x - \log_a y$$
$$\log_a x^n = n \log_a x$$

**Logaritmide vahetus:**
$$\log_a b = \frac{\log_c b}{\log_c a}$$

**Eksponentvõrrandite lahendamine:**
$$a^x = b \Rightarrow x = \log_a b$$

**Logaritmvõrrandite lahendamine:**
$$\log_a x = b \Rightarrow x = a^b$$

**Naturaallogaritm:**
$$\ln x = \log_e x$$

**Kümnendlogaritm:**
$$\lg x = \log_{10} x$$
