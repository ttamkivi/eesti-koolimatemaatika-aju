# 6-nädalane õppeplaan

**Eeldus:** $\sim$1 tund matemaatikat päevas, 5 päeva nädalas. Reaalkooli tasemele sihitud.

## 1. nädal — Arvutamine ja algebra alused

| Päev | Teema | Tegevus |
|---|---|---|
| E | Arvuhulgad, murrud | Teooria + 1.-2. tase ülesanded |
| T | Protsent | Teooria + kõik tasemed |
| K | Astendamine, juurimine | Teooria + 1.-2. tase |
| N | Algebralised avaldised | Teooria + ülesanded |
| R | Korrutamise abivalemid | Teooria + ülesanded + kordamine |

## 2. nädal — Võrrandid

| Päev | Teema | Tegevus |
|---|---|---|
| E | Lineaarvõrrand | Teooria + 1.-2. tase |
| T | Lineaarvõrrand | 3. tase + tekstülesanded |
| K | Võrrandisüsteem | Teooria + asendusmeetod |
| N | Võrrandisüsteem | Liitmismeetod + tekstülesanded |
| R | Ruutvõrrand | Teooria + kõik tasemed |

## 3. nädal — Funktsioonid ja ruutvõrrand

| Päev | Teema | Tegevus |
|---|---|---|
| E | Lineaarfunktsioon | Teooria + graafikud käsitsi |
| T | Lineaarfunktsioon | 3. tase + rakendused |
| K | Ruutfunktsioon | Teooria + tipp, nullkohad |
| N | Ruutfunktsioon | 3. tase + optimeerimisülesanded |
| R | **Proovieksam 1** (90 min) + analüüs |

## 4. nädal — Geomeetria I

| Päev | Teema | Tegevus |
|---|---|---|
| E | Planimeetria | Pindalad: kolmnurk, nelinurgad |
| T | Pythagorase teoreem | Kõik tasemed |
| K | Sarnasus | Teooria + Thales |
| N | Trigonomeetria algus | $\sin, \cos, \tan$ + 30°, 45°, 60° |
| R | Trigonomeetria | 3. tase + rakendused |

## 5. nädal — Geomeetria II ja statistika

| Päev | Teema | Tegevus |
|---|---|---|
| E | Ring ja ringjoon | Teooria + ülesanded |
| T | Stereomeetria | Ruumalad, pindalad |
| K | Statistika, tõenäosus | Keskmine, mediaan, tõenäosus |
| N | Segaülesanded (geom + algebra) | Ümbermõõt/pindala võrrandite kaudu |
| R | **Proovieksam 2** (90 min) + analüüs |

## 6. nädal — Kordamine ja kitsaskohad

| Päev | Teema | Tegevus |
|---|---|---|
| E | Analüüsi proovieksamite vead | Mine tagasi nõrkade teemade juurde |
| T | Tüüpvigade ülevaade | Loe [tüüpvead](../tuupvead/) |
| K | Eksamitehnika | Ajaga tehtud segaülesanded |
| N | Kerge kordamine | Vaata ainult valemid ja mõisted |
| R | **Puhka** — eksam järgmisel nädalal |

## Soovitused

- **Tee kõik ülesanded kirjalikult**, mitte peast — eksamil hinnatakse ka lahenduskäiku.
- **Ole aus** kontrollnimekirja kohta. Ära pea end tugevaks teemas, kus tegelikult eksid.
- **Paranda vead samal päeval** — mitte järgmisel korral.
- **Proovieksam ise ilma kalkulaatorita I osas**, muidu pole päris treening.
