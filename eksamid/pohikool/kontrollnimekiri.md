# Teemade kontrollnimekiri

Iga teema juures on **kriitilised oskused**, mis peavad eksamiks olema automaatsed. Märgi ise: ✅ valdan, ⚠️ vajab kordamist, ❌ pole selge.

## Arvutamine

- [ ] Murdude liitmine, lahutamine, korrutamine, jagamine
- [ ] Kümnend- ja harilike murdude vaheline teisendus
- [ ] Negatiivsete arvudega tehted (märgireeglid)
- [ ] Tehete järjekord ($2+3 \cdot 4 \neq 20$)
- [ ] Protsendi leidmine, terviku leidmine, protsendi arvutamine
- [ ] Astendamine, astmete reeglid ($a^m \cdot a^n = a^{m+n}$)
- [ ] Ruutjuur: $\sqrt{50} = 5\sqrt{2}$, $\sqrt{72} = 6\sqrt{2}$

## Algebra

- [ ] Sulude avamine ($-(2x-3) = -2x+3$)
- [ ] Sarnaste liikmete koondamine
- [ ] Korrutamise abivalemid: $(a \pm b)^2$, $a^2-b^2$
- [ ] Lineaarvõrrandi lahendamine
- [ ] Võrrandisüsteemi lahendamine (asendus või liitmine)
- [ ] Ruutvõrrandi lahendamine (diskriminant)
- [ ] Murdudega võrrand (ühine nimetaja)
- [ ] Tekstülesanne → võrrand

## Funktsioonid

- [ ] Lineaarfunktsiooni graafik ($y = kx + b$)
- [ ] Tõusu leidmine kahe punkti järgi
- [ ] Sirgete lõikepunkt (süsteemina)
- [ ] Ruutfunktsiooni tipp, haarad, nullkohad

## Geomeetria

- [ ] Kolmnurga, ristküliku, rombi, trapetsi, ringi pindala
- [ ] Kolmnurga nurkade summa = 180°
- [ ] Pythagorase teoreem (kaatetid, hüpotenuus)
- [ ] Sarnased kolmnurgad, pindalade suhe ($k^2$)
- [ ] $\sin, \cos, \tan$ täisnurkses kolmnurgas
- [ ] Tähtsad nurgad: 30°, 45°, 60°
- [ ] Ringjoone pikkus $C = 2\pi r$, ringi pindala $S = \pi r^2$
- [ ] Ruumala: kuup, risttahukas, silinder, püramiid, koonus, kera

## Statistika ja tõenäosus

- [ ] Aritmeetiline keskmine, mediaan, mood
- [ ] Tõenäosuse klassikaline definitsioon
- [ ] Vastandsündmus: $P(\bar{A}) = 1 - P(A)$

## Eksamitehnika

- [ ] Oskan eksami läbi lugeda ja ajastada
- [ ] Kirjutan iga lahenduse sammu välja (osaline punkt!)
- [ ] Kontrollin vastust (asendus tagasi)
- [ ] Vastan küsitud vormingus (sh mõõtühikud)
