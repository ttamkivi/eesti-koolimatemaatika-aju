# Eesti Koolimatemaatika Aju 🇪🇪

Avatud lähtekoodiga matemaatika teadmistebaas Eesti koolidele. Katab täieliku õppekava 6.–12. klassini — mõisted, valemid, teooria, harjutusülesanded, eksamiülesanded ja AI-agendid personaalseks õppimiseks.

---

## Mis see on?

See repositoorium on **elav dokument** — mitte lõpetatud õpik, vaid kasvav kogu, kuhu iga õpetaja saab panustada. Selle eesmärk on:

- Olla **selgroog** AI-põhise matemaatikatuutori jaoks
- Olla **jagamisplatvorm** Eesti parimatele matemaatikaõpetajatele
- Olla **abivahend** õpilastele iseseisval õppimisel

---

## Struktuur

```
/oppekava      → Kursuste kaardid koos võtmemõistete, valemite ja õpitulemustega
/teemad        → Teooria selgitused + ülesannete komplektid teemade kaupa
/eksamid       → Põhikooli ja gümnaasiumi eksamiülesanded teemade kaupa
/olumpiaad     → Matemaatikaolümpiaadi juhend, ettevalmistus, näidisülesanded
/opetajale     → 45-min tunni mall + õppevahendid
/agendid       → 3 AI-agenti: tuutor, ülesannete generaator, tunni planeerija
/content       → Hugo veebisaidi sisu (GitHub Pages)
```
