# Korrutamise abivalemid

## 1. Kolm peamist valemit

**Summa ruut:**
$$(a + b)^2 = a^2 + 2ab + b^2$$

**Vahe ruut:**
$$(a - b)^2 = a^2 - 2ab + b^2$$

**Ruutude vahe:**
$$a^2 - b^2 = (a - b)(a + b)$$

## 2. Laiendused

**Summa/vahe kuup:**
$$(a + b)^3 = a^3 + 3a^2 b + 3ab^2 + b^3$$
$$(a - b)^3 = a^3 - 3a^2 b + 3ab^2 - b^3$$

**Kuupide summa ja vahe:**
$$a^3 + b^3 = (a + b)(a^2 - ab + b^2)$$
$$a^3 - b^3 = (a - b)(a^2 + ab + b^2)$$

## 3. Kasutamine

Valemid aitavad:
- kiiresti korrutada
- tegureid välja tuua (faktoriseerida)
- avaldisi lihtsustada
- võrrandeid lahendada

## 4. Lahendatud näited

**Näide 1.** Arvuta $102^2$.
$$102^2 = (100 + 2)^2 = 10000 + 400 + 4 = 10404$$

**Näide 2.** Faktoriseeri $x^2 - 9$.
$$x^2 - 9 = x^2 - 3^2 = (x-3)(x+3)$$

**Näide 3.** Arvuta $99 \cdot 101$.
$$99 \cdot 101 = (100-1)(100+1) = 100^2 - 1 = 9999$$

**Näide 4.** Lihtsusta $\dfrac{x^2 - 6x + 9}{x - 3}$.
$$= \frac{(x-3)^2}{x-3} = x - 3 \quad (x \neq 3)$$

**Näide 5.** Faktoriseeri $4x^2 - 12x + 9$.
$$= (2x)^2 - 2 \cdot 2x \cdot 3 + 3^2 = (2x - 3)^2$$

## 5. Tüüpilised vead

- $(a+b)^2 \neq a^2 + b^2$ — keskmine liige $2ab$ unustatakse.
- Märgi viga vahe ruudus.
- Ruutude summat $a^2 + b^2$ ei saa reaalarvudega faktoriseerida.

- Hulkliikmete tegurdamine.
- Jooned geomeetrias (ringjoone võrrand).
